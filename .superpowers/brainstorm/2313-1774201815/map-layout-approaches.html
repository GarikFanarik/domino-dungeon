A — Fixed Column Grid

All rows share the same column positions. Connections are always clean diagonals. Bottom = start, top = boss. Like Slay the Spire.

B — Path Lanes

3-4 visible vertical "roads" run bottom to top. Nodes sit on lanes, with cross-lane bridges. Clear paths, strong visual lanes. More linear feel.

C — Proportional Coordinates

Each node gets an absolute (x%, y%) based on its column slot and row. No DOM measurement needed — lines always connect to nodes precisely. Keeps the organic feel but fixes the broken lines.

Click to select. My recommendation is A — column grid gives the clearest navigation and is closest to established dungeon-crawler conventions.
